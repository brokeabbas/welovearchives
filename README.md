# Archives Fashion Marketplace

Archives is a multi-vendor fashion marketplace designed for independent Nigerian brands and shoppers.

**Live marketplace:** [welovearchives.com](https://welovearchives.com)

## Product scope

- Product discovery by brand, category, audience, and collection
- Product detail, cart, checkout, and order-confirmation flows
- Customer accounts, profiles, password recovery, and personalised views
- Vendor onboarding and brand dashboards
- Marketplace collections, outfit inspiration, and curated feeds
- Customer contact and live-support integration

## Production stack

React, TypeScript, Node.js, Express, Prisma and MySQL.

## Repository status

This public repository contains a selected frontend application snapshot used to communicate the routing and product scope. The complete production system—including backend services, database schema, operational configuration, and private deployment details—is not published here.

For a deeper technical overview, see the [Archives case study](https://github.com/brokeabbas/software-engineering-portfolio/blob/main/case-studies/archives.md).
